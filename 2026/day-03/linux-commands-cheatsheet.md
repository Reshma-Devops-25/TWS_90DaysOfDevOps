**Process Management**



\- ps :  Display currently running/active processes.

\- ps aux : Displays all running processes.

\- top : Monitor processes in real time.

\- htop : interactive process monitoring tool.

\- kill -9 PID : Force terminate a process.

\- killall proc\_name : Kill all processes by name.

\- pkill process\_name : Kill processes by name.

\- pgrep process\_name : FInd process ID by name.



**File System**



\- pwd : Show present/current working directory.

\- find : Search for files and directories.

\- cd dir : Change directory to directory.

\- cd : Change to home.

\- mkdir -h dir : Create directory

\- rmdir dir : Delete directory.

\- rm file : Delete file

\- rm -f file : Force remove file.

\- rmdir -r dir : Delete directory recursively.

\- cp file1 file2 : Copy file1 to file2.

\- grep pattern file : search text in file.

\- awk '{print $1}' file : Extract specific column.

\- sed 's/old/new' file : Replace text in file.

\- touch file : Creates/ Update file.

\- tail -f logfile.log : Monitor logs in real time.

\- wc -l file.txt : Count lines in file.

\- ln source target : Create hard link.

\- ln -s source target : Create soft link.

\- ls -l | grep ".json" : Lists files and then filters for .json files.



**Networking Troubleshooting**



\- ping : check network connectivity/response.

\- ip addr : Displays IP address information.

\- curl http://google.com : Test website response.

\- dig google.com : DNS lookup.

\- ss -tulpn : Show listening ports.

\- hostnamectl : Display hostname and OS information.



**System Information**



\- date : Show current date and time.

\- cal  : Show this month's calender.

\- uptime : Show current uptime.

\- w : Display who is online.

\- whoami : Who you are logged in as.

\- finger user : Display information about user.

\- uname -a : Show Kernel information.

\- cat /proc/cpuinfo : Show CU information.

\- cat /proc/meminfo : Show Memory information.

\- free -h : Show memory usage.

\- df -h : Show disk usage.

\- du : Show directory space usage.

\- systemctl status service : Check service status.

\- systemctl restart service : Restart a service.
