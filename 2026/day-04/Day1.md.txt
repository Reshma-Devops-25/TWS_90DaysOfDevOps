My understanding of DevOps and Cloud Engineering?

 - DevOps is a combination of Development(Dev) and Operations(Ops).It is a culture and set of practices that help teams to build, test and release software faster with more reliably and scalability.
  Plan -> Devlop -> Build -> Test -> Release -> Deploy -> Operate -> Monitor -> Plan

- Cloud engg is the technical specialization focused on building, managing and securing the scalable infrastructure and platforms(AWS, Azure, GCP)that host these Applikation.

Why is DevOps used?
 - Traditionally development and Operations teams works separately often led to
   > Delays releases
   > Manual errors
   > Communication gaps
   > Slow issue resolution

 -> DevOps helps overcome these challenges through :
   > Automation
   > Continuous Integration and Continuous Delivery(CI/CD)
   > Team collaboration.
   > Faster deployments
   > Improved software quality
   > Better Monitoring and reliability
